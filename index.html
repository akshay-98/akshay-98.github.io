Why Pothole Tracker?

Freedom from bad roads

There are many situations where you leave the house to go to a destination via an unknow route only to know that the road you took was in bad condition which resulted in frustration in the best case and vehicular damage in the worst case, well fret no more, because Pothole Tracker is to the rescue. Using this application you can navigate throught the best routes avaliable to reach destination without hassle or relying on the kindness of strangers.

How it works?

Image recognition and Accelerometer

A user while traveling with the application running helps to gather data for the specific route they are traversing. Data is gathered using the Image recognition software running on the application as well as cross validating it with the accelerometer recordings. This helps gather accurate readings for that route and is stored in a database for future use. The next time another user is traversing the same route this data is served to them so they can make informed decisions regarding the route to select to reach their destination.

Use?

Navigate roads like a pro

There are many uses for this application. Casual users can use this application for their day to day travels without fear that they may encounter bad roads which may cause undue frustration and experience the joys of driving. This can also be used by road authorities to monitor the status of the roads without spending excess budget on collecting data manually.
